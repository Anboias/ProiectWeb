brew install node (for npm and npx commands)

npx create-react-app project (will create new React project called project)

Go to /project
Copy all files from project.zip to /project (Overwrite)

npm install --save json-server
json-server --watch infojudete.json
npm run json:server
npm i -D json-server concurrently
+ made the modification in package.json under scripts to run both the app and the json-server concurrently
